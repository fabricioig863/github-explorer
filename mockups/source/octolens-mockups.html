App mobile · cliente GitHub API · recriação

Octolens.

Um app cliente conceitual que consome a REST API do GitHub — 4 abas principais (Explore, Saved, Feed, Profile) + 3 telas em stack (Detail, Issues, Design System). Linguagem visual própria: tipografia Geist, paleta âmbar-quente, acento violeta-azul. Inspirado no design do GitHub sem clonar a marca.
Arquitetura: 4 tab screens + 3 stack screens
Endpoints: /search/repositories · /repos · /issues · /users/{me}/starred · /users/{me}/events
Stack: React Native + Tanstack Query
Temas: light + dark

Octolens · cliente conceitual da API pública do GitHub · design original.
Tab screens (com bottom nav): Explore · Saved · Feed · Profile · Stack screens (sem bottom nav, push navigation): Repo Detail · Issues · Design System.
7 telas × 2 temas = 14 mockups. Dots de linguagem, ícone de circle-dot para issues e estrutura owner/repo são convenções funcionais — paleta, tipografia, layout e marca são próprios.
